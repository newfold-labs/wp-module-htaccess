# Overview

**wp-module-htaccess** provides centralized .htaccess management for Newfold brand plugins: canonical ruleset, syntax validation, prevention of duplicate/conflicting directives, backups, and a consumer API for other modules (e.g. wp-module-performance) to register rewrite fragments. No runtime Composer requires. Maintained by Newfold Labs. Distributed via Newfold Satis.
